# futu-statement-skill

*[English](README.md) | 中文*

一个 Claude/agent **skill** + 独立 Python 工具，把**富途證券（Futu）香港结单**——
**年度账单（`.xlsx`）** 或 **月结单（`證券月結單` .pdf）**——解析成可直接报税的 CSV，
面向**个人所得税·境外所得（个税 境外所得）**、CRS 申报、券商盈亏对账。

**只读官方结单文件**，不使用富途 OpenD / API（接口给不出已实现盈亏、手续费，分红也不带标签）。

## 为什么需要它

富途港股结单是报税权威凭证，但是：

- **没有「已实现盈亏」栏** —— 必须从成交明细自行重建；
- **没有「派息/股息」栏** —— 现金分红藏在 **`公司行動`（公司行动）** 里；
- 利息、出入金、申赎全混在 **`資金進出`** 一节。

方法沉淀在 [`SKILL.md`](SKILL.md)，并提供脚本自动完成。

## 输入：两种格式，自动识别（xlsx 优先）

| 输入 | 在哪拿 | 说明 |
|---|---|---|
| **年度账单 `.xlsx`**（优先） | 富途 App → 我的 → 賬戶詳情 → 年度账单 | 更干净；带 **期初/期末持仓**，跨年持仓能正确实现 |
| **月结单 `.pdf`**（一份或多份） | 富途 App → 我的 → 賬戶詳情 → 電子結單/月結單 | 需要 `pdftotext`；无期初持仓 |

传文件或传目录都行。**目录里两种都有时，自动选 xlsx**——它更准：月结单只覆盖自然年，
去年 12 月开仓、今年 1 月平掉的持仓在月结单里看不到；年度账单有 `期初` 持仓会把它带进来。
（实测两种口径只差「一笔跨年期权的权利金」。）

## 安装

```bash
pip install openpyxl          # 解析年度账单 .xlsx
brew install poppler          # 解析月结单 .pdf（或 apt install poppler-utils）→ 提供 pdftotext
```

用哪种输入就装哪个。Python 3.10+ 测试通过。

## 用法

```bash
python3 parse_futu_statement.py 2025_年度账单.xlsx -o 输出/ --rate 0.90322   # 年度账单
python3 parse_futu_statement.py /月结单目录/        -o 输出/ --rate 0.90322   # 月结单
python3 parse_futu_statement.py /目录              -o 输出/ --rate 0.90322   # 两者都有→选xlsx
```

`--rate` 可选 —— HKD→RMB 年末中间价；给了就额外输出人民币列。年份自动识别。

### 输出（UTF-8-BOM，Excel 直接打开不乱码）

| 文件 | 内容 |
|---|---|
| `futu_<年>_成交明细.csv` | 股票/期权成交；`变动金额` 已扣手续费 |
| `futu_<年>_股息利息现金流.csv` | 分红（`公司行动`）、利息、出入金、各类费用 |
| `futu_<年>_期权行权到期.csv` | 期权到期（EXP）/ 被行权（ASS） |
| `futu_<年>_已实现盈亏_按标的.csv` | 按标的已实现盈亏（移动加权平均） |
| `futu_<年>_账户净值.csv` | 期初/期末余额（交叉校验） |

脚本会打印 `Σ变动金额`、分红合计、已实现盈亏合计，方便核对。

## 已实现盈亏怎么算

按标的移动加权平均，利用富途的 `開倉/平倉` 标记（买卖方向由 `变动金额` 正负判断，所以
`强平` 强平单也能处理）。到期/被行权的期权按 0 平掉（保费全部计为已实现）；年末仍持有的标的
已实现记 0（属未实现，当年不计税）。年度账单的 `期初` 持仓会带入跨年持仓，故能捕捉跨年项目。
可用 `资产净值变动 − 净出入金` 交叉校验。详见 [`SKILL.md`](SKILL.md)。

## 作为 Claude Code skill 使用

把本目录放进 skills 目录（如 `~/.claude/skills/parsing-futu-statements/`）或用插件管理器安装。
当你让 Claude 解析富途结单、或算富途盈亏/分红用于报税时，会自动加载 [`SKILL.md`](SKILL.md)。

## 隐私

仓库内**不含任何个人数据**。脚本只读你指定的文件、把 CSV 写到本地；`.gitignore` 已拦截
`*.pdf`、`*.xlsx`、`*.csv`，避免误提交结单或结果。

## 可靠性

已用整年数据、两种输入交叉验证：月结单与年度账单在每一笔成交、分红、利息、按标的盈亏上完全
对得上，且年度账单还能多捕捉到一笔月结单看不到的跨年期权。成交数字与代码精确；期权名称由
合约代码反解，股票名称从持仓数据获取。
